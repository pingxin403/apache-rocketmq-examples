# RocketMQ 消息重试与死信队列示例

本示例演示 RocketMQ 5.x 的消息重试机制和死信队列功能。

## 📋 功能说明

- **消息重试**：消费失败后自动重试，最多重试16次
- **死信队列**：重试次数用尽后，消息进入死信队列
- **支付回调场景**：演示支付回调的重试策略
- **订单处理场景**：演示区分业务异常和系统异常

## 🔄 重试机制

| 重试次数 | 重试间隔 | 累计时间 |
|---------|---------|---------|
| 1 | 10秒 | 10秒 |
| 2 | 30秒 | 40秒 |
| 3 | 1分钟 | 1分40秒 |
| ... | ... | ... |
| 16 | 2小时 | 4小时45分40秒 |
