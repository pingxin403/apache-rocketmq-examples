# RocketMQ 批量消息示例

本示例演示 RocketMQ 5.x 的批量消息功能，包括批量发送和批量消费。

## 📋 功能说明

- **批量发送**：一次性发送多条消息，提升吞吐量
- **批量消费**：一次性消费多条消息，提升处理效率
- **智能分批**：自动将消息分批，确保不超过大小限制
- **日志采集**：实战场景，批量采集和发送日志

## 🚀 性能对比

| 方式 | 批量大小 | TPS | 性能提升 |
|------|---------|-----|---------|
| 单条发送 | 1 | 500 | 基准 |
| 小批量 | 10 | 2000 | 4倍 |
| 中批量 | 100 | 10000 | 20倍 |
| 大批量 | 1000 | 20000 | 40倍 |
